# Strategic Analysis Context

## Seed SciCare POV Summary

### Key Seed Perspectives Found

- **Fermented foods ≠ probiotics**: Seed strictly distinguishes Live Dietary Microbes (LDM) from probiotics. Fermented foods may contain live microorganisms but don't meet the scientific criteria for probiotics (defined strain, defined dose, demonstrated benefit). If food sources are discussed, this distinction MUST be maintained. Per SciCare POV: "just because something contains live microorganisms doesn't mean it satisfies the definition of a probiotic."

- **Prebiotics — polyphenol-based vs. fiber-based**: Seed's unique angle is their non-fermenting, polyphenol-based prebiotic from Indian pomegranate (punicalagins). This is already well-covered in the article. SciCare POV reinforces this as a key differentiator vs. competitors using inulin/FOS. Dietary fiber sources should be framed as "small amounts" that support the microbiome broadly, not as replacements for precision-dosed prebiotics.

- **Probiotics are transient, don't colonize**: SciCare POV emphasizes that probiotic strains are generally no longer recoverable 1–4 weeks after intake ceases. Benefits are confined to duration of consumption. Competitors (especially Omni-Biotic) incorrectly claim prebiotics help bacteria "colonize the gut" — Seed would never use this framing. Use "support" or "interact with" instead of "colonize."

### Conflicts Identified

- **Omni-Biotic claims prebiotics help bacteria "colonize the gut"** — contradicts Seed's scientifically-grounded stance that probiotics are transient, not colonizers
- **Competitors casually recommend fermented foods as probiotic sources** — Harvard Health and others treat yogurt/kefir as probiotic equivalents; Seed distinguishes LDM from scientifically-validated probiotics
- **Competitors use "die-off" language** for initial adjustment symptoms — Seed frames this ONLY as "temporary acclimation period," never as "die-off" or "Herxheimer reaction"

### Topics to Handle Carefully

- **No "die-off" or "detox" framing**: If adding acclimation content, Seed frames initial effects as "temporary acclimation period" — NEVER "die-off," "Herxheimer reaction," or "detox"
- **Safety/interactions**: Seed's position is "no known contraindications" between probiotics and other supplements, but always recommend consulting healthcare provider
- **AFU language**: Seed's positioning is that AFU is a "more precise" measurement than CFU (NOT "more accurate")
- **Weight claims**: DS-01 has NOT been studied for weight effects — do not connect prebiotics/probiotics to weight management claims
